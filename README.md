# JobASAP – Mobile Job Matching Application

## Overview
JobASAP is a mobile application designed to help users find flexible and fast job opportunities nearby. The app aims to improve access to short-term employment and supports SDG 8 (Decent Work and Economic Growth).

---

## Features

* User authentication using Firebase
* Browse and view available job listings
* Location-based job search using Google Maps
* Job posting and acceptance system
* Real-time chat between users
* Review and rating system

---

## Technologies Used

* Java & Kotlin
* Firebase Authentication
* Firebase Firestore Database
* Google Maps API
* Android Studio
